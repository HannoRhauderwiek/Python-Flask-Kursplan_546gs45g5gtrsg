Willkommen

In diesem Kurs baut ihr eure eigene Web-App – mit Python und Flask. Keine Vorkenntnisse in Webentwicklung nötig, nur Python-Grundlagen.

Was euch erwartet: In 40 Stunden geht ihr vom ersten HTML-Tag bis zu einer eigenständigen Web-App mit Datenbank, Login und professionellem Design. Jede Stunde wird programmiert – das Web lernt man im Browser, nicht auf Folien.

Was ihr braucht

  • Einen Mac oder PC mit VS Code und der Python-Extension
  • Python 3 (prüfen: Terminal → python3 --version)
  • Einen Browser (Chrome oder Firefox)
  • Internetverbindung (für Bootstrap-CDN)
