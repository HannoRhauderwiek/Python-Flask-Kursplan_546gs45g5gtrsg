Lösungen: Einstieg in Flask

Willkommen zu Block 2! Hier lernst du, deine erste Flask-App zu schreiben. Flask ist ein Python-Framework, mit dem du dynamische Webseiten erstellen kannst. Lass uns loslegen!

Übung 1: Flask erweitern

Deine erste Flask-App mit CSS-Styling und Python-Variablen!

Python-Code (app.py):

from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello():
    name = "Anna"
    return f"""
    <!DOCTYPE html>
    <html lang="de">
    <head>
        <meta charset="UTF-8">
        <title>Meine Flask-App</title>
        <style>
            body {{
                font-family: Arial, sans-serif;
                background-color: #f0f0f0;
                text-align: center;
                padding: 20px;
            }}
            h1 {{
                color: #333;
                font-size: 3em;
            }}
            p {{
                color: #666;
                font-size: 1.2em;
            }}
        </style>
    </head>
    <body>
        <h1>Hallo {name}! 👋</h1>
        <p>Willkommen zu deiner ersten Flask-App!</p>
        <p>Du bist am {datetime.now().strftime('%d.%m.%Y')} hier.</p>
    </body>
    </html>
    """

if __name__ == '__main__':
    app.run(debug=True)

Wichtig: Die doppelten Klammern {{ und }} sind notwendig, damit Python die CSS-Klammern nicht als f-String interpretiert!

Übung 2: Steckbrief-Generator

Eine dynamische Route, die Tiere darstellt! Mit verschiedenen URLs kannst du verschiedene Tierinfos anzeigen.

Python-Code (app.py):

from flask import Flask

app = Flask(__name__)

# Tierdaten
tiere = {
    'loewe': {
        'name': 'Löwe',
        'emoji': '🦁',
        'info': 'Der König der Tiere! Löwen sind kraftvoll und majestätisch.'
    },
    'elefant': {
        'name': 'Elefant',
        'emoji': '🐘',
        'info': 'Der größte Landsäugetier! Elefanten sind intelligent und sozial.'
    },
    'pinguin': {
        'name': 'Pinguin',
        'emoji': '🐧',
        'info': 'Ein Vogel, der nicht fliegen kann, aber hervorragend schwimmt!'
    },
    'schildkroete': {
        'name': 'Schildkröte',
        'emoji': '🐢',
        'info': 'Langsam aber sicher! Schildkröten können sehr alt werden.'
    }
}

@app.route('/')
def home():
    return """
    <!DOCTYPE html>
    <html lang="de">
    <head>
        <meta charset="UTF-8">
        <title>Steckbrief-Generator</title>
        <style>
            body {{ font-family: Arial; text-align: center; padding: 20px; }}
            h1 {{ color: #333; }}
            ul {{ list-style: none; padding: 0; }}
            li {{ margin: 10px; }}
            a {{
                text-decoration: none;
                background-color: #007bff;
                color: white;
                padding: 10px 20px;
                border-radius: 5px;
            }}
            a:hover {{ background-color: #0056b3; }}
        </style>
    </head>
    <body>
        <h1>🐾 Steckbrief-Generator 🐾</h1>
        <p>Wähle ein Tier:</p>
        <ul>
            <li><a href="/steckbrief/loewe">Löwe</a></li>
            <li><a href="/steckbrief/elefant">Elefant</a></li>
            <li><a href="/steckbrief/pinguin">Pinguin</a></li>
            <li><a href="/steckbrief/schildkroete">Schildkröte</a></li>
        </ul>
    </body>
    </html>
    """

@app.route('/steckbrief/<tier>')
def steckbrief(tier):
    if tier not in tiere:
        return "<h1>Tier nicht gefunden! 😢</h1>"

    t = tiere[tier]
    return f"""
    <!DOCTYPE html>
    <html lang="de">
    <head>
        <meta charset="UTF-8">
        <title>Steckbrief: {t['name']}</title>
        <style>
            body {{
                font-family: Arial;
                background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
                color: white;
                text-align: center;
                padding: 20px;
                min-height: 100vh;
            }}
            .card {{
                background-color: rgba(255, 255, 255, 0.1);
                padding: 30px;
                border-radius: 10px;
                max-width: 500px;
                margin: 0 auto;
            }}
            .emoji {{ font-size: 4em; margin-bottom: 20px; }}
            h1 {{ font-size: 2.5em; }}
            p {{ font-size: 1.2em; line-height: 1.6; }}
            a {{
                display: inline-block;
                margin-top: 20px;
                background-color: #fff;
                color: #667eea;
                padding: 10px 20px;
                text-decoration: none;
                border-radius: 5px;
                font-weight: bold;
            }}
            a:hover {{ background-color: #f0f0f0; }}
        </style>
    </head>
    <body>
        <div class="card">
            <div class="emoji">{t['emoji']}</div>
            <h1>{t['name']}</h1>
            <p>{t['info']}</p>
            <a href="/">← Zurück</a>
        </div>
    </body>
    </html>
    """

if __name__ == '__main__':
    app.run(debug=True)

So funktioniert die Route: - /steckbrief/<tier> ist eine dynamische Route - der Teil nach / wird als Variable übergeben - Mit tier hast du Zugriff auf das, was der Nutzer eingegeben hat - Das Dictionary tiere speichert alle Informationen - Mit if tier not in tiere: checkst du, ob das Tier existiert

Übung 3: Portfolio mit Templates

Jetzt nutzen wir Jinja2-Templates, um unseren Code sauberer zu machen!

Python-Code (app.py):

from flask import Flask, render_template, url_for

app = Flask(__name__)

@app.route('/')
def home():
    return render_template('index.html')

@app.route('/ueber-mich')
def about():
    info = {
        'name': 'Anna Mueller',
        'alter': 17,
        'stadt': 'Berlin',
        'hobby': 'Programmieren'
    }
    return render_template('about.html', info=info)

@app.route('/projekte')
def projects():
    projekte = [
        {'name': 'To-Do App', 'beschreibung': 'Eine einfache Aufgaben-App', 'status': 'Fertig'},
        {'name': 'Chat-Bot', 'beschreibung': 'Ein einfacher Chat-Bot in Python', 'status': 'In Arbeit'},
        {'name': 'Blog', 'beschreibung': 'Mein persönlicher Blog', 'status': 'Geplant'}
    ]
    return render_template('projects.html', projekte=projekte)

@app.route('/kontakt')
def contact():
    return render_template('contact.html')

if __name__ == '__main__':
    app.run(debug=True)

Wichtige Konzepte: - {% extends "base.html" %} erbt von der Basis-Template - {% block content %}...{% endblock %} sind Platzhalter für unterschiedliche Inhalte - { url_for('home') } erzeugt automatisch die richtige URL zur angegebenen Funktion - {% for %}...{% endfor %} ist eine Schleife in Jinja2 - Diese Struktur macht deinen Code viel übersichtlicher!

Übung 4: Klassenliste Challenge

Eine Tabelle mit Schülerdaten, Farben basierend auf Noten, und Jinja2-Filter!

Python-Code (app.py):

from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def klassenliste():
    schueler = [
        {'name': 'Anna Mueller', 'mathe': 1, 'deutsch': 2, 'englisch': 1},
        {'name': 'Ben Schmidt', 'mathe': 3, 'deutsch': 3, 'englisch': 2},
        {'name': 'Clara Weber', 'mathe': 2, 'deutsch': 1, 'englisch': 2},
        {'name': 'David Braun', 'mathe': 4, 'deutsch': 3, 'englisch': 4},
        {'name': 'Eva Klein', 'mathe': 1, 'deutsch': 2, 'englisch': 1},
    ]
    return render_template('klassenliste.html', schueler=schueler)

if __name__ == '__main__':
    app.run(debug=True)

Template: templates/klassenliste.html

<!DOCTYPE html>
<html lang="de">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Klassenliste</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
    <style>
        .note-1 { background-color: #d4edda; } /* Grün für 1 */
        .note-2 { background-color: #fff3cd; } /* Gelb für 2 */
        .note-3 { background-color: #ffe5e5; } /* Rosa für 3 */
        .note-4 { background-color: #f8d7da; } /* Rot für 4+ */
    </style>
</head>
<body class="p-4">
    <div class="container">
        <h1>📚 Klassenliste</h1>
        <p>Gesamt: {{ schueler|length }} Schüler</p>

        <table class="table table-striped table-hover">
            <thead class="table-dark">
                <tr>
                    <th>#</th>
                    <th>Name</th>
                    <th>Mathe</th>
                    <th>Deutsch</th>
                    <th>Englisch</th>
                </tr>
            </thead>
            <tbody>
                {% for s in schueler %}
                <tr>
                    <td>{{ loop.index }}</td>
                    <td>{{ s.name }}</td>
                    <td>
                        <span class="note-{{ s.mathe if s.mathe <= 4 else 4 }}">
                            {{ s.mathe }}
                        </span>
                    </td>
                    <td>
                        <span class="note-{{ s.deutsch if s.deutsch <= 4 else 4 }}">
                            {{ s.deutsch }}
                        </span>
                    </td>
                    <td>
                        <span class="note-{{ s.englisch if s.englisch <= 4 else 4 }}">
                            {{ s.englisch }}
                        </span>
                    </td>
                </tr>
                {% endfor %}
            </tbody>
        </table>
    </div>
</body>
</html>

Erklärung der Jinja2-Features: - { schueler|length } zählt die Schüler (das ist ein Filter) - { loop.index } gibt die Nummer in der Schleife aus (1, 2, 3, …) - class="note-{{ s.mathe }}" nutzt die Note, um die CSS-Klasse zu bestimmen - Der if ... else Teil sorgt dafür, dass auch Noten > 4 rot werden - {% for s in schueler %}...{% endfor %} iteriert über alle Schüler
