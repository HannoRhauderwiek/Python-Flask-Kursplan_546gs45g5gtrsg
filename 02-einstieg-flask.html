2  Einstieg in Flask

Willkommen zu Flask! In diesem Kapitel lernst du, wie du mit Python eine Website mit mehreren Seiten erstellst – viel einfacher, als du vielleicht denkst. Lass uns gemeinsam starten!

2.1 UE 4: Hello World mit Flask

2.1.1 Was ist Flask?

Flask ist ein Micro-Framework für Python, das dir hilft, Webanwendungen zu bauen. Es ist leicht, schnell und perfekt zum Lernen. Mit Flask kannst du:

  • Webseiten mit Python erstellen
  • Verschiedene URLs (Routen) definieren
  • Mit Daten aus Formularen arbeiten
  • Später sogar Datenbanken verbinden

Flask ist ein Python-Framework für Webseiten. Es übernimmt die komplizierte Arbeit im Hintergrund, damit du dich auf das Wichtige konzentrierst: deine Website!

2.1.2 Python schnell auffrischen

Bevor wir Flask nutzen, schauen wir uns kurz an, was du über Python wissen solltest:

Variablen und Datentypen:

# Strings (Text)
name = "Alice"
greeting = f"Hallo, {name}!"

# Listen
früchte = ["Apfel", "Banane", "Orange"]
print(früchte[0])  # Apfel

# Dictionaries (Wörterbücher mit Schlüsseln)
person = {"name": "Bob", "alter": 25, "stadt": "Berlin"}
print(person["name"])  # Bob

# Zahlen
alter = 16
preis = 9.99

Funktionen und Return-Werte:

Das Wichtigste für Flask: Funktionen, die etwas zurückgeben!

def begruessung(name):
    nachricht = f"Guten Tag, {name}!"
    return nachricht

ergebnis = begruessung("Maria")
print(ergebnis)  # Guten Tag, Maria!

Das return-Statement ist super wichtig! In Flask schreiben wir Funktionen, die HTML zurückgeben – das funktioniert genau wie return in dieser Auffrischung.

2.1.3 Flask installieren

Öffne dein Terminal und gib ein:

pip3 install flask

Schau nach, dass es funktioniert hat. Du solltest sehen, dass Flask installiert wurde. Super!

2.1.4 Dein erstes Flask-Programm

Erstelle eine neue Datei app.py und schreibe diese Zeilen:

from flask import Flask

app = Flask(__name__)

@app.route('/')
def startseite():
    return '<h1>Hallo Welt!</h1>'

if __name__ == '__main__':
    app.run(debug=True)

Jetzt starten wir die App im Terminal:

python3 app.py

Du solltest etwas Ähnliches sehen:

 * Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)
 * Debugger is active!

Öffne deinen Browser und gehe zu http://127.0.0.1:5000/ – und dort ist deine Website! 🎉

Tipp

Was bedeutet 127.0.0.1? Das ist “localhost” – der Computer, auf dem du arbeitest. 5000 ist der Port (eine Art Tür) für deine Website.

2.1.5 Das Code-Geheimnis erklärt

Schauen wir uns Zeile für Zeile an, was passiert:

Zeile Was macht’s?
from flask import Flask Wir laden Flask
app = Flask(__name__) Wir erstellen die Flask-Anwendung (die Setup-Zeile)
@app.route('/') Das ist der Wegweiser: Diese Funktion läuft, wenn jemand / besucht
def startseite(): Die Funktion hat einen Namen
return '<h1>Hallo Welt!</h1>' Wir geben HTML zurück – das zeigt der Browser
app.run(debug=True) Wir starten die Website; debug=True hilft bei Fehlern

Der @app.route(‘/’)-Decorator ist ein “Wegweiser”. Er sagt Flask: “Wenn jemand diese URL besucht, laufe die darunter stehende Funktion!”

2.2 UE 5: Routing – Mehrere Seiten, eine App

Jetzt wird’s spannend: Wir bauen eine Website mit mehreren Seiten!

2.2.1 Statische Routen

Im Moment haben wir nur eine Route (/). Lass uns mehr hinzufügen:

from flask import Flask

app = Flask(__name__)

@app.route('/')
def startseite():
    return '<h1>Startseite</h1><p>Willkommen!</p>'

@app.route('/ueber')
def ueber():
    return '<h1>Über uns</h1><p>Wir sind ein cooles Team.</p>'

@app.route('/kontakt')
def kontakt():
    return '<h1>Kontakt</h1><p>Email: info@example.com</p>'

if __name__ == '__main__':
    app.run(debug=True)

Jetzt kannst du zu diesen URLs gehen: - http://127.0.0.1:5000/ - http://127.0.0.1:5000/ueber - http://127.0.0.1:5000/kontakt

Wichtig: Jede Funktion braucht einen unterschiedlichen Namen! startseite(), ueber(), kontakt() – alle verschieden. Sonst verstehen sich Flask nicht!

2.2.2 Dynamische Routen – Mit Variablen

Das Beste kommt jetzt: Routen mit Variablen! So kannst du auf <name> zugreifen:

@app.route('/hallo/<name>')
def hallo(name):
    return f'<h1>Hallo {name}!</h1>'

Probier aus: - http://127.0.0.1:5000/hallo/Alice - http://127.0.0.1:5000/hallo/Bob - http://127.0.0.1:5000/hallo/Dein-Name

Hinweis

Dynamische Routen sind wie Schablonen. Der Teil in < > wird zu einer Variable, die die Funktion erhält. Cool, oder?

2.2.3 Navigation mit url_for()

Wenn wir mehrere Seiten haben, brauchen wir Links. Flask hat eine smarte Funktion dafür:

from flask import Flask, url_for

app = Flask(__name__)

@app.route('/')
def startseite():
    ueber_link = url_for('ueber')  # Erzeugt den Link zur ueber-Seite
    kontakt_link = url_for('kontakt')

    html = f'''
    <h1>Startseite</h1>
    <a href="{ueber_link}">Über uns</a> |
    <a href="{kontakt_link}">Kontakt</a>
    '''
    return html

url_for() ist dein Freund! Es sorgt dafür, dass Links immer stimmen – selbst wenn du die URLs später umbenennst. Flask aktualisiert die Links automatisch.

2.2.4 Häufige Fehler

# ❌ FALSCH: Funktionsname vergessen
@app.route('/bio')
def biografie():  # ← Der Name!
    return '<h1>Bio</h1>'

# Wenn du dann url_for('bio') schreibst, funktioniert's nicht!
# Richtig: url_for('biografie')

# ❌ FALSCH: Slash vergessen
@app.route('kontakt')  # Kein / am Anfang!
def kontakt():
    return 'Kontakt'

# ❌ FALSCH: Parameter vergessen
@app.route('/gruss/<name>')
def gruss():  # ← name-Parameter fehlt!
    return f'Hallo {name}'

2.3 UE 6: Jinja2-Templates

2.3.1 Das Problem mit HTML in Python

Bis jetzt schreiben wir HTML als lange Strings in Python – das wird schnell chaos:

@app.route('/klassenliste')
def klassenliste():
    html = '''
    <html>
    <head><title>Klassenliste</title></head>
    <body>
    <h1>Unsere Klasse</h1>
    <table>
    <tr><th>Name</th><th>Note</th></tr>
    <tr><td>Alice</td><td>1.5</td></tr>
    <tr><td>Bob</td><td>2.0</td></tr>
    </table>
    </body>
    </html>
    '''
    return html

Das ist chaos! Und der Editor versteht das HTML nicht richtig.

Warnung

Das ist nicht der Weg. HTML sollte nicht in Python-Strings versteckt sein. Wir brauchen eine bessere Lösung: Templates!

2.3.2 Jinja2-Templates – Die Lösung

Flask nutzt Jinja2, eine Template-Sprache. Das ist HTML mit superkräften!

  1. Erstelle einen Ordner templates neben deiner app.py
  2. Darin erstellst du .html-Dateien
  3. In Python lädst du die Templates mit render_template()

Dateistruktur:

flask-kurs/
├── app.py
└── templates/
    └── startseite.html

2.3.3 Template-Variablen: {{ }}

Hier ein einfaches Template:

templates/startseite.html:

<!DOCTYPE html>
<html>
<head>
    <title>Willkommen</title>
</head>
<body>
    <h1>Hallo {{ name }}!</h1>
    <p>Dein Alter: {{ alter }}</p>
</body>
</html>

app.py:

from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def startseite():
    return render_template('startseite.html', name='Alice', alter=16)

if __name__ == '__main__':
    app.run(debug=True)

Jetzt wird aus { name } Alice und aus { alter } 16!

{{ }} ist die Syntax für Variablen in Jinja2. Alles dazwischen wird durch Python-Werte ersetzt.

2.3.4 Loops in Templates: {% for %}

Oft brauchst du Listen! Dafür nutzt du {% for %}:

templates/fruchtliste.html:

<h1>Meine Früchte</h1>
<ul>
{% for frucht in fruechte %}
    <li>{{ frucht }}</li>
{% endfor %}
</ul>

app.py:

@app.route('/')
def startseite():
    meine_fruechte = ['Apfel', 'Banane', 'Orange', 'Erdbeere']
    return render_template('fruchtliste.html', fruechte=meine_fruechte)

Ausgabe:

Meine Früchte
- Apfel
- Banane
- Orange
- Erdbeere

2.3.5 Bedingungen in Templates: {% if %}

Mit {% if %} kannst du HTML nur unter bestimmten Bedingungen anzeigen:

<h1>Noten</h1>
{% if note <= 2.0 %}
    <p style="color: green;">Sehr gut!</p>
{% elif note <= 3.0 %}
    <p style="color: orange;">Gut.</p>
{% else %}
    <p style="color: red;">Bitte lernen!</p>
{% endif %}

Template-Syntax: - { variable } – Zeige einen Wert - {% for item in liste %}...{% endfor %} – Schleife - {% if bedingung %}...{% endif %} – Bedingung

2.3.6 Schleifenvariable: loop.index

Manchmal brauchst du die Nummer des aktuellen Schleifendurchlaufs:

<table>
{% for name in schueler %}
    <tr>
        <td>{{ loop.index }}</td>
        <td>{{ name }}</td>
    </tr>
{% endfor %}
</table>

Das gibt loop.index aus: 1, 2, 3, …

2.3.7 Filter: |length, |upper, |lower

Jinja2 hat auch Filter (mit | geschrieben):

<h1>Klassenliste ({{ schueler|length }} Schüler)</h1>
<p>Alle Namen in Großbuchstaben:</p>
<p>{{ schueler_liste|upper }}</p>
Tipp

Nützliche Filter: - |length – Länge einer Liste - |upper – Großbuchstaben - |lower – Kleinbuchstaben - |reverse – Umgekehrte Reihenfolge

2.3.8 Komplexes Beispiel: Klassenliste

Hier ein realistisches Beispiel mit Schüler-Daten:

templates/klassenliste.html:

<!DOCTYPE html>
<html>
<head>
    <title>Klassenliste</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
<div class="container mt-5">
    <h1>Klassenliste 12B</h1>
    <p>Insgesamt {{ schueler|length }} Schüler</p>

    <table class="table table-striped">
    <thead>
        <tr>
            <th>#</th>
            <th>Name</th>
            <th>Durchschnitt</th>
            <th>Status</th>
        </tr>
    </thead>
    <tbody>
    {% for schueler in schueler_liste %}
        <tr>
            <td>{{ loop.index }}</td>
            <td>{{ schueler.name }}</td>
            <td>{{ schueler.schnitt }}</td>
            <td>
            {% if schueler.schnitt <= 2.0 %}
                <span class="badge bg-success">Sehr gut</span>
            {% elif schueler.schnitt <= 3.0 %}
                <span class="badge bg-warning">Gut</span>
            {% else %}
                <span class="badge bg-danger">Schwach</span>
            {% endif %}
            </td>
        </tr>
    {% endfor %}
    </tbody>
    </table>
</div>
</body>
</html>

app.py:

from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def klassenliste():
    schueler = [
        {'name': 'Alice', 'schnitt': 1.5},
        {'name': 'Bob', 'schnitt': 2.3},
        {'name': 'Charlie', 'schnitt': 3.2},
        {'name': 'Diana', 'schnitt': 1.8},
    ]

    return render_template('klassenliste.html', schueler_liste=schueler)

if __name__ == '__main__':
    app.run(debug=True)

Das gibt eine schöne Tabelle mit: - Nummern (loop.index) - Namen und Durchschnitten - Farbigen Badges (gut/schwach) basierend auf der Note - Bootstrap-Styling

2.4 Zusammenfassung

Du hast gerade drei große Dinge gelernt:

  1. Hello World mit Flask – Wie eine Website mit Python startet
  2. Routing – Mehrere Seiten mit verschiedenen URLs
  3. Templates – Sauberes HTML mit Jinja2-Superkräften

Das Wichtigste zum Mitnehmen:

# 1. Flask importieren
from flask import Flask, render_template

# 2. App erstellen
app = Flask(__name__)

# 3. Routen definieren mit @app.route()
@app.route('/seite/<parameter>')
def meine_funktion(parameter):
    # 4. Templates rendern
    return render_template('template.html', daten=parameter)

# 5. Starten
app.run(debug=True)
